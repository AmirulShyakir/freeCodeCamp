Overview

Shakstetic is an alias that the Java and web developer Amirul Shyakir goes by on social media accounts such as Github and TikTok. It is a play on his last name and the word aesthetic, a nod to his interests in creating aesthetically pleasing yet functional designs. Additionally, his interests lies in fitness, where the word aesthetic describes a movement and way of life of those involved in the gym culture.

In this documentation, Amirul Shyakir's life would be explored further through API calls in pseudocode.

Installation

In order to install the latest beta of Amirul Shyakir (Amirul_Shyakir_23.0), please type the following text into the terminal:

npm install amirul_shyakir --v23.0

To run Amirul Shyakir in runtime environment, type the following in the terminal:

npm run amirul_shyakir

Sleeping Positions

In Amirul_Shyakir_23.0, there are various sleeping positions to configure. These include "plank", "hand-pillow", "protector", "cat-spread". The following code block shows an example of a typical sleep session.

while (sleep) { hand-pillow(); plank(); protector(); if (alarm == true) { break; } }

The following example code shows how to change the default sleeping position.

settings { sleep-position-default: "plank"; }
